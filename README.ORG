** How to test \ install
Установить python 3

На windows: winget install python3

На arch базированных дистрибутивах: pacman -S python3

На Debian базированных дистрибутивах: apt install python3

На Fedora\Red Hat базированных дистрибутивах: dnf install python3

Установить данные библиотеки для python по средством комманды pip install <название пакета>:

asyncio

aiogram

aiohttp

requests

Перейти на https://mgkeit.space/developers#home, создать собственный API ключ и вставить его а 723 строчку

Запустить main.py

Бот находится по данному username @Chariton_bot

** How to test \ install
Установить python 3

Установить данные библиотеки для python:

asyncio

aiogram

aiohttp

requests

Перейти на https://mgkeit.space/developers#home, создать собственный API ключ и вставить его а 348 строчку

Запустить main.py

Бот находится по данному username @Chariton_bot

** Logging documentation
usercommandrequests.txt - файл логов статичных комманд по типу /help, /timetable

Buildingslogs.txt - файл логов выбора филиалов

Groupslogs.txt - файл логов выбора групп

userrequests.txt -  файл логов остальных запросов пользователя
